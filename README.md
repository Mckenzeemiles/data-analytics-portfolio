Data Analytics Portfolio
This repository contains end-to-end data analysis projects covering dataset understanding, data cleaning, exploratory data analysis (EDA), visualization, and insight generation — built using Python (pandas, matplotlib, seaborn) and SQL (MySQL Workbench).
Projects
1. Netflix Titles Analysis
An analysis of Netflix's movie and TV show catalog metadata.
What's included:
Dataset understanding (structure, data types, primary key identification)
Data cleaning (missing values, duplicates, format standardization, validation)
Exploratory data analysis (content type distribution, ratings, genres, top countries)
Visualizations (bar charts, line charts, pie charts, histograms, box plots)
Key insights on content trends and catalog composition
Files:
Netflix_Data_Analysis_Complete.ipynb — full notebook (Tasks 1–5)
netflix_titles_cleaned.csv — cleaned dataset
Netflix_Summary_Report.docx — one-page summary report
2. Online Retail Analysis
An analysis of transactional sales data from a UK-based online retailer.
What's included:
Dataset understanding (structure, data types, composite key identification)
Data cleaning (missing CustomerIDs, duplicates, encoding issues, returns handling)
Exploratory data analysis (top products, revenue by country, monthly sales trends, customer behavior)
Visualizations (bar charts, line charts, pie charts, histograms, box plots)
Key insights on sales patterns and customer purchasing behavior
Files:
OnlineRetail_Data_Analysis_Complete.ipynb — full notebook (Tasks 1–5)
OnlineRetail_cleaned.csv — cleaned dataset
Tools & Technologies
Python: pandas, numpy, matplotlib, seaborn
SQL: MySQL Workbench
Environment: Jupyter Notebook
